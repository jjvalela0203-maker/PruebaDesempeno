# Students Register 

## Description 📖

This is a python simple program 
Allows users to manage a list of students
The register is  modularized to ensure a clean code structure

## 🚀 Features

* Add new students
* Show the list of the actual students
* search for a student by a specific characteristic (ID, Name, Age, Program, State)
* Update the data of one student
* Delete a student from the list

## ▶️ How to Run

1. Clone this repository or download the files:

```bash
git clone <your-repo-url>
cd <your-project-folder>
```

2. Run the program:

```bash
python app.py
```

## Examples of use

This program can be use for example in a school, being used to register students and keep track of their activity at school
